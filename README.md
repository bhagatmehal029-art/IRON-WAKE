### **THE SYSTEM HAS CHOSEN YOU**

**IRONWAKE** isn’t just another fitness app — it’s a **gamified training protocol** wrapped in a sleek cyberpunk HUD interface. Every rep, every plank, every day you show up — the System watches, records, and rewards.

### ✨ Core Features
- **Cinematic Boot Sequence** with typewriter narrative and atmospheric HUD design  
- **Age-Adaptive Training System** — scales difficulty for ages 8–90+  
- **Real-time Camera Rep Detection** (Push-ups, Sit-ups, Squats) using motion analysis  
- **Plank Timer** with live progress tracking  
- **Daily Quests + Streak System** — maintain your Hunter streak  
- **RPG Progression** — Level up, gain stats (STR, VIT, AGI), earn XP  
- **Manual Override** — + / − buttons for accuracy  
- **PWA Ready** — installable on mobile, works offline  
- **Premium Dark Cyber Aesthetic** with glowing cyan/violet accents  

### 🎯 Perfect For
- Fitness beginners seeking motivation  
- Parents looking for fun family training  
- Gamers who want to level up in real life  
- Anyone who loves immersive tech + self‑improvement  

### 🛠 Tech Stack
- HTML5 + CSS3 + Vanilla JavaScript  
- Camera API + Canvas motion processing  
- Local Storage (no backend required)  
- Fully responsive & mobile‑first  
- Progressive Web App (PWA) support
